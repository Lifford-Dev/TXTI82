How to make a web page with txti

The only thing you have to give txti is the "content".Txti takes care of the rest,but you
can take advantage of these option:

Custom URL

This is the part goes after"txti.es/"like".es/barry". Currently only accepts
Lowercase letter(0-9),and dashes (-). if you put anything else in
there will change it.

Custom Edit Code

Txti will give you a random edit code,but you can changes it.You have to have this to
make changes to the txti,and it can never ever be retrieve if lost.

Title

Give your txti a relevant title so it shows up better when shared on socail media
search results and in browser window

Author

Lets people know who 'made this!This shows up in some searches and socail media
posts and your twitter handle work really nicely if you post the link on twitter.

Discription

This is a short(200 character)summary of the page.This shows up in twitter cards
and search results.Txti will automatically use the first 200 characters of your txti as a
description if you don't provide one.

Make your txti easy to read and understand

You can make txti robust and full featured with links,images,lists,headings and
more.Txti uses a popular set of rules called Markdown

  • #Heading 1=biggest heading

    • ##Heading 2=second biggest heading

    ##Heading 2=second biggest heading

  • Two spaces(spaces bars)at the end of a line starts a new line
  • Return twice(leaving an empty line) starts a new paragraph
  • *itallics*=italics
  • **Bold**=bold
  • links[link to txti](http://txti.es)=link to txti(note:be sure to include the link to txti part of the link )
  • Images*:![Monkey selfie](http://i.imgur.com/FXSBf8c.jpg)
  • Bulleted lists:

    • -bulleted item a

    bulleted item b

    -bulleted item c

  • Numbered lists:

    • 1.Numbered item 1

    2.Numbered item 2

    3.Numbered item 3

    *Images must be uploaded somewhere else(we recommended imgur.com )Because
    images can really slow down pages and we're all about"fast web pages for everybody,"
    images are not displayed by default.The reader has the option of displaying the images in
    the page or just viewing links to them.see an example txti.es/images

    Important notes

  • Txti will delete any pages that are more than 6 months old but have been viewed
    less than two times ever.If you make a txti and forget about it,it might not be there 6
    months later.
  • Later concerns or pages you think violate txti's terms of servicecan be sent via
    private message or tweet to @txties on twitter.
